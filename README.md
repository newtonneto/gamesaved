# Configurações para rodar o projeto

## Ambiente

```
  node: v16.13.1
  yarn: 1.22.10
  nvm: 0.39.1
  java: 11.0.10
```

## Executar

```sh
  $ yarn start
```

```sh
  $ yarn android
  or
  $ yarn ios
```
## Environment

Criar um arquivo .env com as seguintes chaves na raiz do projeto.

```
  GAMEAPI_KEY=your_key_here
  GAMEAPI_BASE_URL=https://api.rawg.io/api/
```
